Kalahari bacteria

Microbiome analysis of biological soil crusts

Download this project as a .zip file Download this project as a tar.gz file

Niche partitioning of bacterial communities in biological crusts and soils under grasses, shrubs and trees in the Kalahari

David R Elliott, Andrew D Thomas, Stephen R Hoon, Robin Sen (2014)

Biodiversity and Conservation. April 2014

Record of analysis

The data and code in this repository were used to perform the analyses presented in the above paper, specifically the release tagged as accepted version. Later versions are provided for compatibility with new software versions and presentation adjustments etc.

Output

The main output is the file kalahari_BSC_bacteria.html but many other files are also generated. The most important ones are directly linked in the html file, and the others can be found by browsing the gh-pages branch of this repository.
